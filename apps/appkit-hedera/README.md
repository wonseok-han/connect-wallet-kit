# Getting Started

* Appkit 라이브러리로 Hedera 지갑을 연결해 이더리움 트랜잭션 기반으로 전송 트랜잭션이 정상 동작하는지 확인하기 위한 초간단 앱

### 실행

```bash
# Module Install
pnpm install

# Server start
pnpm run dev
```
